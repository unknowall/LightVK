## LightVK a Vulkan C# low-level bindings. 

This repository contains low-level bindings for Vulkan.<br>
This binding is generated from vk.xml (https://github.com/KhronosGroup/Vulkan-Docs/blob/main/xml/vk.xml).

#### Features
 - Vulkan 1.0, 1.1, 1.2, 1.3, 1.4
 - All Vulkan Extensions as Vulkan RayTracing
 - For Windows, Linux, Android

#### Used in PS1 emulator ScePSX:
 - https://github.com/unknowall/ScePSX
